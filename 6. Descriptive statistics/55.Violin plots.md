#### Code
데이터 생성
```python
## create the data

n = 1000
thresh = 5 #threshhold for cropping data

data = np.exp(np.random.randn(n))
data[data>thresh] = thresh + np.random.randn(sum(data>thresh))*.1
plt.plot(data,'+')
```
![[55.gen_data.png]]

히스토그램 생성
```python
# show histogram
plt.hist(data,30)
plt.title('Histogram')
plt.show()

# show violin plot
plt.violinplot(data)
plt.title('violn')
plt.show()
```
